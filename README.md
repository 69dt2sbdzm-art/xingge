# 🕵️ Who's the Spy? - English Edition

一个支持多人实时联机的"谁是卧底"英语版网页游戏。

## 游戏特色
- 🎮 支持 3-8 人同时在线游戏
- 📱 手机/平板/电脑都能玩
- 🌍 中英双语词汇，边玩边学英语
- ⚡ 实时同步，无需刷新

## 部署步骤（如需自建）

### 步骤 1：创建 GitHub 仓库
1. 访问 https://github.com/new
2. 仓库名称：`xingge`（或你喜欢的名字）
3. 选择 Public（公开）
4. 点击 Create repository

### 步骤 2：上传文件
1. 点击 "uploading an existing file"
2. 上传 `index.html` 文件
3. 点击 Commit changes

### 步骤 3：启用 GitHub Pages
1. 进入 Settings → Pages
2. Source 选择 "Deploy from a branch"
3. Branch 选择 "main"，文件夹选 "/ (root)"
4. 点击 Save，等待 1-2 分钟

### 步骤 4：访问游戏
地址格式：`https://你的用户名.github.io/仓库名`
