# GitHub Pages 部署步骤

## 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名称：`whos-the-spy-game`
3. 选择 **Public**（公开）
4. 点击 **Create repository**

## 步骤 2：上传文件

在仓库页面：
1. 点击 **"uploading an existing file"**
2. 上传 `index.html` 文件
3. 点击 **Commit changes**

## 步骤 3：启用 GitHub Pages

1. 进入仓库的 **Settings** 标签
2. 左侧点击 **Pages**
3. **Source** 选择 "Deploy from a branch"
4. **Branch** 选择 "main"，文件夹选 "/ (root)"
5. 点击 **Save**
6. 等待 1-2 分钟

## 步骤 4：访问游戏

地址格式：`https://你的用户名.github.io/whos-the-spy-game`

例如：`https://xingge19.github.io/whos-the-spy-game`
